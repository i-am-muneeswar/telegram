# Project Name - Telegram

## Abstract
This Project is like telegram and here we will implement 8 functionalities like
1. Sign-UP
    - Create Profile
2. Sign-IN
    - Delete Profile
    - View Profile
    - View All Profiles
        - Sort by Name
        - Sort by Address
    - Edit Profile
    - Search Profile
        - Sort by Name
        - Sort by Address
    - Timeline
